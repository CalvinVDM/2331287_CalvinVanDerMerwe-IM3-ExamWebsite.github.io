Week 11 - Decolonising Technology pt2 / Responsive Design

How we code matters was an interesting lecture. When we look at code and its syntax, I always like the idea of how logically thought out and planned it is done to me it represents a way of thinking that can literally construct anything digitally. Code is literally the language of computers, and it is a language that we created and build upon to manifest our imaginations digitally.

I realise that with code whatever we think if we can boil it down to a logical process, we can create it and redefine it as we please and that is something that is interesting to me. Coding is complicated and needs to be easy to understand if we what to communicate and build with each other. With code, we must use our logic and simplify it to its simplest form so that others can see and use this code as well as that it is something that’s difficult. It is also worth noting that there are many ways of coding something and that sometimes the way one person makes it isn’t the best way it can be done.

Responsive design was cool when thinking about how we should make things to be better and act better so that its more interactive and it does something based on the user’s preferences.
