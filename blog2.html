Week 2 - Internet Histories / HTML Fundamentals

I already have first-hand experience from High school with using HTML and I enjoyed using it then although we used it to do the framework as well as the stylish look for the website. HTML is a much similar side to a website, and I found that getting to know its rules and syntax is very easy and pleasing. Using HTML has become second nature so I'm more excited to better understand using CSS and Java.

Looking at the reading "Why You Should Choose HTML5" BY Lawson, Bruce was really interesting because it was very informative on the basics and starting points of using HTML. Looking at the reading with fresh eyes made me see how much easier HTML can be when it is explained in the way the reading does. the reading makes the knowledge very digestible and easy to understand and if I had to learn HTML from scratch this would be very useful for achieving that.

The next reading "How A Screen Reader User Accesses The Web: A Smashing Video." by Lawson, Bruce was again very informative and easy to understand especially in the way that it explains navigation. this made me rethink the way I was doing nav in my older websites and it helps to look at how other websites do navigation with these examples, I think it would be easier to look at a newer version of navigation on my websites.

Internet histories were really interesting to look at and hear about but I was not particularly interested in the lecture because I have gone through it before I enjoyed remembering the talks we had about this and how the growth of the internet was influenced throughout the years.
